SendSync is made specifically to avoid any and all servers when sending things between peers.

It uses very tight security, and you should never share your passwords with anybody, even if they are trusted.

NEVER tell ANYONE except your peer what your shared key is. NEVER send the shared key over unencrypted messaging software.

1. To use SendSync, put in a shared key from your peer, and generate a code.
2. Send the code to your peer, and have them send you theirs.
